信義區

吳興街商圈

以白天的傳統市場最為集中，晚上是零星的夜市小吃，其他的店家則多為商店型態， 包括許多美食餐館與醫療用品店，但由於吳興街的街道不寬而且錯綜複雜，找起路來要費點心思， 這片區域中有許多特色餐飲，值得前往細細品嘗。

坐落位置 : 約為吳興街165~269號，包括220巷、莊敬路325巷

鄰近捷運 : R03 台北101/世貿站

商圈類型 : 傳統市場、餐飲店、醫療相關用品、飲品

信義計畫區百貨商圈

在不知不覺中正在漸漸取代東區，成為大臺北地區最具指標性的時髦都會區； 白天的信義計畫區百貨商圈是一個生活節奏忙碌又快速的商業金融中心， 夜晚則成為充滿豔麗光彩奪目的時髦都會場所， 一到週末假日又會變成超大型舞台，經常舉辦各種假日活動或園遊會。

坐落位置 : 約為信義路五段、松仁路、松高路及市府路所圍區域

鄰近捷運 : BL18 市政府站

商圈類型 : 大型百貨、影城
