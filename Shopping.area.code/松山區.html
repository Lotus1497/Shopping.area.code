松山區

民生社區商圈

民生社區商圈以民生東路與三民路交叉之圓環為中心，沿街商店住宅分佈於民生東路、三民路、新中街、 撫遠街、新東街、民權東路主要幹道。店家類型多元完整，包含知名咖啡廳、特色餐廳、文創小店、 補習班及才藝班等，屬於優質的社區型商圈，也因社區內公共建設完善，食、衣、住、行、育、樂一應俱全， 讓社區生活資源更加豐富多源，此外民生社區公園綠地眾多，綠蔭空間寬廣，創造出優質之生活環境品質。

坐落位置 : 東至基隆河（現行最接近路段為塔悠路），西至敦化北路， 南至延壽街和敦化北路199巷連線，北至松山機場，行政區包括了松山區的精忠里、東昌里、介壽里、東榮里、 三民里、富泰里、富錦里、新東里、新益里、莊敬里等十個里

鄰近捷運 : BR13 松山機場站

商圈類型 : 食、衣、住、行、育、樂

南京復興捷運站周邊商圈

以捷運南京東路站為中心，這裡的氛圍輕鬆愜意，少了擁擠的人潮，多了生活的閒適，有點偷得浮生半日閒的味道。 最醒目的建築，莫過於先施百貨了；轉個彎，兄弟大飯店，可見金髮碧眼的外國人穿梭其中。 走向另一頭，慶城街上則坐落著許多特色超人氣餐廳； 不遠處的皇冠出版社及行天宮圖書總館，則增添了不少的文化氣息。

坐落位置 : 約為民生東路四段、敦化北路及南京東路三段、復興北路所圍區域

鄰近捷運 :　G16 / BR11 南京復興站

商圈類型 : 餐飲小吃、商務飯店、百貨量販
