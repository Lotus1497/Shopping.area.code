南港車站CITYLINK周邊商圈

南港車站為「三鐵共構」之一，不僅交通便利，生活機能亦非常健全。 此外，位於車站附近的CITYLINK百貨也是大型的車站購物商城， 有別於一般車站給人只是購買紀念品和填飽肚子的中繼站的感覺， 以「Shopping Resort」為概念，提供顧客一個陽光、慢活、駐足、購物的悠雅空間， 商城內也結合了幾米的泡泡裝置藝術，是個讓人能充分五感體驗的地方。

坐落位置 : 約為市民大道八段、興中路及忠孝東路七段之間

鄰近捷運 : BL22 南港站

商圈類型 : 餐飲、百貨量販、休閒

中國信託金融園區商圈

位於捷運南港軟體園區站附近的中國信託金融園區是中國信託的企業總部，為複合式商辦大樓。 園區內分為A、B、C三棟，以「家」為概念，象徵著爸爸、媽媽、小孩相互依靠扶持的家庭觀。 除了餐廳之外，園區內亦設置文薈館，可使人深入了解中國信託的發展歷史， 此外還有以「公益」為出發點的滑冰場，常常邀請弱勢、偏鄉孩童體驗滑冰的樂趣。

坐落位置 : 約為經貿二路及三重路附近

鄰近捷運 : BR23 南港軟體園區站

商圈類型 : 餐廳會館、休閒
