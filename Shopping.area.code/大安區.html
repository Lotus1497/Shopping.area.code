東區商圈

東區位於重要交通要道上，擁有便利的交通運輸規劃與設施， 且結合潮流藝文元素及夜經濟之特性以及巷弄特色店家成為創店首發站， 並成為發展「夜食」、「夜遊」、「夜享」、「夜讀」及「夜時尚」之「東區五夜城」。 不論是以年輕族群為主的運動品牌，高價位的各國精品店也能在各個路口彰顯， 無數種新奇創意的美食亦藏身在巷弄之間，因而吸引許多年輕人及上班族群在此聚集， 另亦為臺北市時尚流行之重鎮。

坐落位置 : 北至市民大道、東至光復南路、西至復興南路、南至仁愛路所圍區域

鄰近捷運 : 捷運BL15/BR10 忠孝復興站

商圈類型 : 以服飾、百貨、特色餐廳及連鎖餐飲為主

永康商圈

以「永康公園」為中心，大多以特色餐飲為主，早期發展與鼎泰豐息息相關， 先是由於媒體報導讓鼎泰豐成為國際觀光客旅遊的知名美食之一，也帶動起永康街的知名度。 接著小小的巷道開始聚集一家家臺灣風味的餐館，之後更加入許多異國特色美食和風格獨具的咖啡小館， 慢慢地許多個性小舖也紛紛在此聚集，逐漸形成濃厚的人文風格，也成為永康街的獨特魅力， 並成為聞名國際的觀光景點。

坐落位置 : 約為北市信義路二段和金山南路二段(雙側)及和平東路一段和新生南路二段(單側)

鄰近捷運 : 捷運R07/O06 東門站

商圈類型 : 以特色餐飲、古董藝品及服飾為主
